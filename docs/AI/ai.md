AI LEARNING ROADMAP

Phase 1 — Foundation (1–3 months)
Math: Linear algebra, probability, stats, calculus
Python: NumPy, pandas, matplotlib
Concepts: ML types (supervised, unsupervised, RL)

Phase 2 — Machine Learning (2–3 months)
Algorithms: Linear/logistic regression, decision trees, SVMs, clustering
Library: Scikit-learn
Practice: Kaggle datasets

Phase 3 — Deep Learning (2–4 months)
Topics: Neural networks, CNNs, RNNs, Transformers
Frameworks: PyTorch or TensorFlow

Phase 4 — Specialization (pick one)
- NLP / LLMs
- Computer Vision
- Reinforcement Learning
- Generative AI

Phase 5 — Applied & Production (ongoing)
MLOps, MLflow, Hugging Face, build projects on GitHub

Plan 1: Talk to your Database (Text-to-SQL Agent)
This is called a Text-to-SQL or NL-to-SQL system. You don't need to build a model from scratch at all.
What you actually need to learn:

Python basics
How LLM APIs work (OpenAI / Anthropic)
Prompt engineering — teaching the model your DB schema
LangChain or LlamaIndex (agent frameworks)
Basic SQL knowledge (you need it, even if users don't)

BEGINNER AI PROJECTS — LEARNING BY DOING

LEVEL 1 — Easiest to Start (Week 1–2)
• Chatbot with personality → API calls, prompt engineering
• PDF summarizer → File handling, LLMs
• YouTube transcript summarizer → Data extraction, summarization
• Resume analyzer → Document parsing, prompting

LEVEL 2 — A Bit More Involved (Month 1–2)
• News sentiment analyzer → NLP, classification
• Expense tracker with AI → Data + LLM combo
• Voice-to-text note taker → Audio APIs, Whisper
• FAQ chatbot for a website → Embeddings, RAG basics

LEVEL 3 — Closer to Your Goals (Month 2–3)
• CSV chat / ask your Excel → Direct prep for DB plan
• Stock price dashboard + alerts → Direct prep for trading plan
• Simple buy/sell signal bot → Indicators, backtesting basics
• Mini SQL agent on a small DB → Foundation for Plan 1
