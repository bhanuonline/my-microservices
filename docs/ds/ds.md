### DATA STRUCTURES AND ALGORITHMS - COMPLETE LIST

===========================================
DATA STRUCTURES
===========================================

LINEAR DATA STRUCTURES:
- Array (fixed size, contiguous memory)
- Dynamic Array (ArrayList in Java)
- Linked List (Singly, Doubly, Circular)
- Stack (LIFO - Last In First Out)
- Queue (FIFO - First In First Out)
- Deque (Double-ended Queue)
- Priority Queue (Heap-based)

NON-LINEAR DATA STRUCTURES:
- Binary Tree
- Binary Search Tree (BST)
- AVL Tree (self-balancing BST)
- Red-Black Tree (self-balancing BST)
- B-Tree and B+ Tree
- Heap (Min Heap, Max Heap)
- Trie (Prefix Tree)
- Segment Tree
- Fenwick Tree (Binary Indexed Tree)
- Graph (Directed, Undirected, Weighted)

HASH-BASED STRUCTURES:
- Hash Table / Hash Map
- Hash Set
- LinkedHashMap
- LinkedHashSet

ADVANCED STRUCTURES:
- Disjoint Set (Union-Find)
- Suffix Array
- Suffix Tree
- Skip List
- Bloom Filter
- LRU Cache (using HashMap + Doubly Linked List)

===========================================
ALGORITHMS
===========================================

SORTING ALGORITHMS:
- Bubble Sort - O(n²)
- Selection Sort - O(n²)
- Insertion Sort - O(n²)
- Merge Sort - O(n log n)
- Quick Sort - O(n log n) average
- Heap Sort - O(n log n)
- Counting Sort - O(n+k)
- Radix Sort - O(d×n)
- Bucket Sort - O(n+k)
- Shell Sort - O(n log n)

SEARCHING ALGORITHMS:
- Linear Search - O(n)
- Binary Search - O(log n)
- Jump Search - O(√n)
- Interpolation Search - O(log log n) average
- Exponential Search - O(log n)
- Ternary Search - O(log₃ n)

GRAPH ALGORITHMS:
- Breadth-First Search (BFS)
- Depth-First Search (DFS)
- Dijkstra's Algorithm (shortest path)
- Bellman-Ford Algorithm (shortest path with negative weights)
- Floyd-Warshall Algorithm (all pairs shortest path)
- Kruskal's Algorithm (Minimum Spanning Tree)
- Prim's Algorithm (Minimum Spanning Tree)
- Topological Sort
- Tarjan's Algorithm (strongly connected components)
- Kosaraju's Algorithm (strongly connected components)
- A* Search Algorithm
- Cycle Detection (using DFS/BFS/Union-Find)

TREE ALGORITHMS:
- Tree Traversal (Inorder, Preorder, Postorder, Level-order)
- Lowest Common Ancestor (LCA)
- Diameter of Tree
- Height/Depth of Tree
- Binary Search Tree operations (insert, delete, search)
- Tree Balancing (AVL rotations, Red-Black operations)
